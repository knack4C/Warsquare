Warsquare er et spil hvor det gælder om at slå alle fjendens tropper. Spillet
har indbygget hjælp og UI der forklare spillets regler. Spillet er lavet som
eksamensprojekt i programmering på HTX.

Projektet lærte os meget om UI, spiludvikling, opsætning og implementering af 
nye ideer under udviklingsprocessen.

Spillet kan ikke køre hvis sprites mappen, ikke er i den samme som .exe filen
Kildekoden kan også findes i 'Warsquare.py' eller 'warsquare.txt'
